Run `mnemosyne --help` and summarize the available commands for the user, grouped by: storing memories (remember, mine, sweep, import), retrieval (search, wake-up), knowledge graph (kg), organization (drawer, tunnel, hallways, taxonomy), security (vault, verify, backup, repair), and serving (serve-mcp, serve-http, daemon).
